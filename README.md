## 🏗 ソフトウェア構成（MVCモデル）
本プロジェクトでは保守性と可読性を高めるため、MVCモデルを採用して実装しました。

- **Model (Bean, DAO)**: データの保持および、MySQLとのデータベース通信を担当。
- **View (JSP)**: Servletから受け取ったデータを元に、ユーザー画面を動的に生成。
- **Controller (Servlet)**: ユーザーからのリクエストを受け取り、適切なビジネスロジックを実行してViewへ制御を転送。
